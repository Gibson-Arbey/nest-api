# NEST API
1. Clonar proyecto
2. ```npm i```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Crear la base de datos
6. Correr el proyecto con el comando: ```npm run start:dev```
7. Ejecutar SEED 
```http://localhost:3000/api/seed```
